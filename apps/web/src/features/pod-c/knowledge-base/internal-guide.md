# 🏢 WorkPresso 전사 운영 가이드 (Official)

본 문서는 WorkPresso의 공식 운영 및 기술 표준 가이드라인입니다.

## [개발 표준]
- **프레임워크**: Next.js 14.2.13 (App Router)
- **상태 관리**: TanStack Query (React Query)
- **UI 라이브러리**: Tailwind CSS + shadcn/ui
- **브랜치 전략**: Git Flow 준수 (feat/, fix/, docs/)

## [복리후생]
- **식대 지원**: 법인카드로 점심 식대(1.2만 원 한도) 및 야근 시 석식 전액 지원.
- **교육비 지원**: 직무 관련 도서 구매 및 유료 강의 비용 무제한 지원.
- **카페 테리아**: 무제한 에스프레소, 스낵 및 신선한 과일 상시 제공.
- **장비 지원**: 최신형 MacBook Pro 및 허먼밀러 의자 기본 제공.

## [예외 및 주의사항]
- **긴급 이슈**: 서비스 장애 발생 시 코어 타임 외에도 즉시 슬랙 채널(#emergency)을 통해 공유해야 합니다.
- **외부 유출 금지**: 내부 지식 베이스 및 디자인 시스템 자산의 무단 외부 공유를 엄격히 금지합니다.
- **보안 위반**: 개인 노트북에 프로덕션 DB를 덤프하거나 기밀 데이터를 저장하는 행위는 경고 없이 징계 대상이 될 수 있습니다.
- **코드 리뷰**: 모든 PR은 최소 2명의 리뷰어 승인을 얻어야 머지 가능하며, 긴급 배포 시에도 사후 리뷰를 반드시 수행해야 합니다.

## [근무 환경]
- **코어 타임**: 13:00 ~ 17:00 (필수 협업 시간)
- **리모트 근무**: 주 2회 권장
- **회의 원칙**: 모든 회의는 30분 이내 종료, 아젠다 필수 공유

## [보안 수칙]
- 모든 API 키 및 비밀값은 `.env.local`에서 관리하며 Git 커밋을 엄격히 금지합니다.
- 프로덕션 데이터베이스 접근은 인가된 IP에서만 가능합니다.
